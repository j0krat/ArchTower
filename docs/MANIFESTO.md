# ArchTower Manifesto

ArchTower existe para construir una workstation Arch Linux confiable, cómoda y mantenible.

No busca reemplazar Arch Linux.
Busca acompañarlo.

## Principios

### Seguridad antes que velocidad

ArchTower no debe ejecutar acciones peligrosas sin avisar.

### Transparencia

Cada comando debe mostrar qué hace y por qué lo hace.

### Automatizar sin esconder

ArchTower puede automatizar tareas repetitivas, pero nunca debe ocultar la complejidad al usuario.

### Comodidad con propósito

Cada función debe resolver una necesidad real.

### Cero basura

ArchTower no debe generar archivos innecesarios ni ensuciar el sistema.

### Confianza

El usuario debe sentir que ArchTower ayuda, no que toma el control.

## Usuario objetivo

ArchTower está pensado para "j0krat" y para cualquier persona que quiera una experiencia Arch Linux más confiable, cómoda y documentada.
